1. rounded corners look nicer on buttons, start using border-radius (not sure on input boxes though)
2. box-shadow (outside is default) use ‘inset’ for inside box shadow.

  • Shadow outside makes button (or input) look higher than the rest of the document
  • Shadow inset makes button (or input) look lower than the rest of the document
